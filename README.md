# Cricket Club Website

Welcome to the Cricket Club Website project! This project is designed to provide a comprehensive online platform for the Adamstown Cricket Club, featuring essential information and functionalities for both members and visitors.

## Project Structure

The project is organized into the following main directories:

- **src**: Contains the source code for the application.
  - **app**: The main application files, including pages for Home, About Us, Contact Us, and Members.
  - **components**: Reusable components such as Header, Footer, and forms for member registration and login.
  - **lib**: Utility functions for API calls and authentication.
  - **hooks**: Custom hooks for managing application state and logic.
  - **styles**: Global CSS styles for the application.
  - **types**: TypeScript interfaces and types used throughout the application.
- **server**: Contains backend logic, including controllers, services, and database schema.
- **tests**: Unit and integration tests for components and functionalities.
- **mobile-sdk**: API specifications for mobile app integration.
- **.env.example**: Example environment configuration file.
- **package.json**: Project dependencies and scripts.
- **tsconfig.json**: TypeScript configuration.
- **next.config.js**: Next.js configuration settings.
- **tailwind.config.js**: Tailwind CSS configuration.
- **postcss.config.js**: PostCSS configuration.

## Features

- **Home Page**: Displays information fetched from the Adamstown Cricket Club website.
- **About Us**: Provides details about the club's history, mission, and values.
- **Contact Us**: Allows users to reach out to the club for inquiries and support.
- **Members Page**: Includes options for new member registration and existing member login.
- **Responsive Design**: Built with mobile compatibility in mind, ensuring a seamless experience across devices.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd cricket-club-website
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` to view the application.
